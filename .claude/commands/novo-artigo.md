## 1. Frontmatter

Gere o frontmatter YAML com os campos abaixo. A data de publicação deve ser a data de hoje.

- `title`: título atrativo e SEO-friendly em português, máximo 60 caracteres
- `description`: descrição de 150–160 caracteres para SEO e para o card do blog
- `pubDate`: data de hoje no formato YYYY-MM-DD
- `author`: "Técnico Carlos Silva"
- `tags`: array de 3–5 tags relevantes (exemplos: "iPhone", "Bateria", "Reparo", "Samsung", "MacBook", "Emergência")

## 2. Conteúdo

Escreva um artigo completo em português brasileiro com:

- **Introdução** (1–2 parágrafos contextualizando o problema para o leitor)
- **3–5 seções** com `##` headers abordando: sintomas, causas, o que fazer, dicas de prevenção
- Use listas numeradas ou com marcadores, negrito para termos-chave e formatação markdown
- Tom técnico mas acessível, voltado a usuários comuns em Recife/PE
- Inclua uma seção sobre como a Conecta Phone resolve o problema, com detalhes do serviço (prazo, garantia, diferenciais)

## 4. Nome do arquivo

Escolha um nome de arquivo em kebab-case baseado no título, curto e descritivo.

Exemplos:
- "Tela do iPhone trincada? Saiba o que fazer" → `tela-iphone-trincada.md`
- "Bateria do Samsung descarregando rápido" → `bateria-samsung-descarregando.md`
